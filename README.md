# vPLAAF 虚拟中国空军网站项目

## 📋 项目概述

vPLAAF 虚拟中国空军是一个以飞行模拟为核心的虚拟航空组织网站。本项目旨在展示组织特色、提供加入渠道，并明确免责声明，确保用户了解该组织的虚拟性质。

## ✈️ 项目特点

- 🎨 **专业设计**：采用军事蓝+金色主题配色，体现空军专业感
- 📱 **响应式布局**：适配桌面、平板和移动设备
- ⚡ **高性能**：轻量级代码，快速加载
- 🎭 **丰富内容**：包含特色介绍、入门要求、免责声明等
- 🔗 **多入口**：多个QQ群加入按钮，方便用户参与
- 🚧 **施工页面**：包含专业的施工中页面

## 🗂️ 文件结构

```
vPLAAF-website/
├── index.html                    # 主网站页面
├── under_construction.html       # 施工中页面
├── style.css                     # 主要样式文件
└── img/                          # 图片资源文件夹
    └── Logo.png                  # 网站Logo（可选添加）
```

## 🚀 快速开始

### 本地运行

1. **下载项目文件**
   ```bash
   git clone [项目仓库地址]
   ```

2. **打开网站**
   - 双击 `index.html` 文件在浏览器中打开
   - 或使用本地服务器（如Live Server）运行

3. **查看施工页面**
   - 双击 `under_construction.html` 查看施工中页面

### 部署到服务器

1. **上传文件**：将所有文件上传到您的Web服务器
2. **配置域名**：将域名指向服务器IP地址
3. **访问网站**：通过域名访问网站

## 🎨 自定义内容

### 修改文本内容

直接编辑HTML文件中的文字内容：

- **主标题和描述**：在 `index.html` 的 `<section class="hero">` 部分
- **特色介绍**：在 `features-section` 部分
- **入门要求**：在 `requirements-section` 部分
- **免责声明**：在 `disclaimer-section` 部分

### 修改链接

1. **QQ群链接**：
   ```html
   将以下链接替换为您的实际QQ群链接：
   https://qm.qq.com/q/cQI7AV3d3q
   
   位置：
   - 导航栏：第28行
   - 英雄区域：第46行
   - 入门要求部分：第145行
   - 页脚：第154行
   ```

2. **Logo图片**：
   ```html
   如需添加Logo，将Logo.png放入img文件夹
   然后在导航栏添加：
   <img src="./img/Logo.png" alt="Official Logo" style="height: 50px">
   ```

### 修改样式

编辑 `style.css` 文件中的CSS变量来更改颜色主题：

```css
:root {
  --primary-dark: #0a2342;     /* 深蓝 - 主色调 */
  --primary-medium: #1a3a5f;   /* 中蓝 */
  --accent-gold: #c9a227;      /* 金色点缀 */
  --accent-green: #12b886;     /* QQ按钮绿色 */
  /* ... 其他变量 */
}
```

## 🔧 开发指南

### 添加新页面

1. 创建新的HTML文件
2. 使用相同的CSS样式表
3. 保持一致的导航栏和页脚结构
4. 确保响应式设计

### 修改样式

1. **全局样式**：在 `:root` 变量中修改
2. **组件样式**：找到对应的CSS类名进行修改
3. **响应式样式**：在对应的媒体查询中调整

### 添加新功能

1. **JavaScript**：如需交互功能，可创建 `script.js` 文件
2. **新组件**：在CSS中添加对应的样式类
3. **新页面**：参考现有页面结构创建

## 🚧 施工页面

当网站部分内容正在建设时，可以使用 `under_construction.html`：

1. **启用方式**：将施工页面重命名为 `index.html`
2. **自定义**：修改进度百分比、预计完工时间等信息
3. **恢复**：完成后恢复原始 `index.html`
